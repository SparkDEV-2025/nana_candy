# Nana Maidala's Candy Shop

### Project Overview
A professional landing page for an authentic northern Nigeria local candy business based in Jama'are Local Government. 

### Features
* **Responsive Design:** Optimized for both mobile and desktop viewing.
* **Product Showcase:** Highlights traditional local candies.
* **Local Identity:** Specifically designed to represent the authentic taste of Jama'are.

### Technologies Used
* **HTML5**
* **CSS3**

### How to View
Simply open the `index.html` file in any web browser to see the live design.
